Heute gibts mal wieder ein Codeschnipsel das Anfänger wohl zum Verzweifeln treiben würde.
Die folgende, syntaktisch als richtig gewertete Befehlsfolge steht zur Diskussion:


$blah="tröööt";
if($blah != 0)
{
    if($blah=="tröööt")
    {
        echo "Sie haben getrötet!";
    } else {
        echo 'Sie haben "'.$blah.'" gesagt.';
    }
} else {
    echo "Sie haben eine 0 eingegeben";
}

Was wird PHP wohl ausgeben? Wer die Antwort findet bekommt einen Gummipunkt!
Richtig! Es ist "Sie haben eine 0 eingegeben". Der Grund ist ebenfalls offensichtlich, "tröööt" ist keine Zahl.
