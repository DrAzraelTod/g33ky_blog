Javascript, wo parseInt('09') ohne Fehler zu 0 aufgelöst wird -> http://tinyurl.com/6dgd6q

Merke: Javascript nimmt an, dass alles was mit einer 0 anfängt eine Oktalzahl ist und wenn Ziffern vorkommen, die in Oktalzahlen nicht vorkommen können (8,9) dann wird einfach dennoch angenommen dass das eine Oktalzahl ist. Natürlich könnte man dann einen Fehler auswerfen, aber einfach 0 zurückliefern ist doch viel einfacher. Nur ein Grund unter 1000en warum Javascript nicht grade meine Lieblingssprache ist und es auch niemals wird.
