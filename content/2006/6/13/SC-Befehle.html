Hier die in alpha-060609a implementierten Befehle

allgemeine Channel-Befehle:

  • !date oder !datum: gibt das aktuelle Datum nach dem Diskordianischen Kalender aus. (mit gregorianischem Datum als Parameter auch beliebige andere)
  • !wiki: liefert die URL zu einem dt. Wikipedia-Artikel zurück (Parameter erforderlich) (bei mehren Wörtern atm stark verbuggt)
  • !google: liefert die URL zu einer Google-Suche mit den Parametern als Suchtext zurück
  • !blog: liefert die URL zu diesem Blog zurück
  • !addfact: fügt Fakten zur Datenbank hinzu (funzt atm nicht)
  • !join: Beitritt zu einem anderem Channel (Parameter)
  • !rot13: (de)codiert den als Parameter übergebenen Text mit Rotate13
  • !trivia: liefert einen Zufälligen Fakt des trivialen wissens
  • !flirt: spricht den als Parameter angegebenen User mit einem zufälligem Flirtspruch an
  • !quote: zitiert eine Zufällige Zeile aus dem Channel-log
  • !warquote oder !wq: Beleidigt den als Parameter angegebenen User mit einem Zitat
normale Query-Befehle:
  • !help: Gibt eine Befehlsliste mit kurzen erklärungen aus (veraltet). Mit einem Befehl als Parameter wird eine Hilfe zu diesem Befehl ausgegeben
  • !flirt sollte den angegebenen per /msg ansprechen und anflirten (funzt atm nicht)
  • !blog: gibt die URL zu diesem Blog aus
  • !date oder !datum: gibt das Diskordianische Datum aus
  • !sw4: liefert ein Zitat aus Star Wars - Episode 4
  • !warquote oder !wq: liefert ein beleidigendes Zitat
  • !wiki: liefert die URL zu einem Wikipedia-Artikel (funzt im Gegensatz zu der Channel Version o-O)
  • !google: liefert die URL zu einer Google-Suche (funzt im Gegensatz zu der Channel Version o-O)
  • !addfact: Fügt einen Fakt zur Datenbank hinzu (atm nicht verfügbar)
  • !rot13: wandelt einen String mit Rotate13 um
Administrator Channel-Befehle:
  • !hop: rejoint den channel
  • !kick: kickt den user (Parameter)
  • !ban: bannt den user (Parameter)
  • !bankick oder !bk: kick und ban (parameter)
